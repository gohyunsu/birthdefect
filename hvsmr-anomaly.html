Research plan · HVSMR-2.0

Define anomaly before training it.

HVSMR-2.0은 CHD cohort입니다. 따라서 이 계획은 “normal vs CHD”와 “CHD cohort 내부의 structural outlier”를 하나의 task로 섞지 않고, 정상 reference를 확보한 뒤 patient-level anomaly detection을 검증하는 순서로 설계합니다.

Decision first

HVSMR alone cannot establish a normal-vs-CHD detector.

60명 모두 CHD patient이므로, HVSMR만으로 학습한 score는 “일반 인구에서의 congenital anomaly probability”가 아닙니다. 특히 mild는 정상 해부학에 가깝거나 수술 후 anatomy가 복원된 case를 포함하지만, healthy control이라는 뜻은 아닙니다.

따라서 최종 primary endpoint는 matched normal CMR control과 CHD CMR을 patient level에서 구분하는 structural anomaly score로 둡니다. HVSMR은 CHD 구조·segmentation·hard-case validation을 담당하고, normal reference는 별도 cohort가 담당합니다.

Primary clinical claim

Normal-reference anomaly detection

정상 대조군에서 정상 구조 분포를 학습하고, hold-out CHD에서 얼마나 벗어나는지 평가합니다. 이 claim에는 matched normal controls가 필수입니다.

Secondary feasibility claim

Within-CHD structural outlier score

mild subgroup을 임시 reference로 삼아 severe 또는 복잡 anatomy를 우선 검토 대상으로 ranking합니다. 이는 CHD detection이 아니라 cohort 내부 triage입니다.

Not a valid claim

“HVSMR alone detects CHD”

healthy population과의 comparison이 없으므로 이 주장은 하지 않습니다. severity는 prognosis가 아니라 anatomy deviation category입니다.

Reference data

What a usable normal cohort must contain.

정상 CMR은 단지 “질환 코드가 없는 image”가 아닙니다. 적어도 구조적으로 정상이라는 임상 판독 기준, CHD cohort와 비교 가능한 3D whole-heart coverage, acquisition metadata, patient-level identifier가 필요합니다. 가능하면 8 structure label을 직접 확보하고, 불가능하면 held-out subset에서 수동 annotation으로 mapping error를 측정합니다.

Required · primary

Matched clinical controls

동일 또는 유사 scanner·sequence·contrast·age range의 structurally normal CMR을 IRB/DUA 아래 확보합니다. controls는 development/validation/test로 patient-level 분리하고, test subset에는 8-structure manual review를 수행합니다.

Role: normal reference + primary evaluation
Conditional · bridge

UK Biobank CMR

대규모 cardiac MRI와 chamber segmentation/normal reference 연구 기반은 유용하지만, access 승인·restricted platform이 필요하고 adult cine protocol과 HVSMR의 pediatric 3D CMR은 다릅니다.

Role: representation pretraining / external shift analysis
Conditional · robustness

MM-WHS MRI

whole-heart MR benchmark지만 label ontology가 7 structures이고 SVC/IVC가 없으며, HVSMR-2.0 논문도 일부 MR case가 CHD임을 지적합니다. 정상 대조군으로 가정하지 않습니다.

Role: label-mapped robustness test only
Go / no-go. matched normal controls를 확보하지 못하면 primary endpoint를 “normal-vs-CHD detection”으로 등록하지 않습니다. 그 경우 연구는 HVSMR 내부 structural outlier ranking과 segmentation robustness로 한정하고, 임상 screening claim을 제거합니다.

Pre-registered workflow

Four phases, each with a decision gate.

Phase 0 · data contract

Harmonize before modeling

  • normal/CHD cohort의 protocol, age, sex, contrast, spacing, field of view를 table로 고정
  • 8-structure ontology와 missing-structure policy를 문서화
  • patient-level split을 먼저 versioning

Gate: control cohort의 protocol mismatch가 크면 image-intensity model을 보류하고 segmentation-derived geometry branch부터 시작

Phase 1 · structure reliability

Train the segmentation backbone

  • 3D U-Net/nnU-Net-style baseline과 topology-aware variant 비교
  • CHD severity·age·acquisition subgroup별 오류 분석
  • RV/RA absent label은 presence-aware metric으로 분리

Gate: held-out structure quality가 불안정하면 predicted mask 기반 anomaly score를 primary로 사용하지 않음

Phase 2 · normal reference

Build interpretable structural scores

  • structure volume, pairwise ratio, centroid distance, vessel/chamber connectivity, shape embedding 추출
  • training controls만으로 robust z-score·Mahalanobis·one-class model fit
  • age/sex/body-size/protocol conditional reference를 validation에서 선택

Gate: normal validation에서 score drift가 크면 threshold를 고정하지 않고 harmonization을 재검토

Phase 3 · patient validation

Evaluate clinical utility, not slice accuracy

  • hold-out patient에서 normal vs CHD, subgroup별 severe/mild ranking 평가
  • fixed review budget에서 recall과 false-positive burden 산출
  • 오류 case를 score·mask·structure descriptor와 함께 판독

Gate: external/protocol-shift test가 없으면 general clinical deployment claim을 하지 않음

Evaluation protocol

Measure each layer with the metric that matches it.

anomaly detection의 최종 단위는 3D volume 한 건, 즉 patient-level입니다. slice·patch를 독립 표본으로 세어 confidence interval을 부풀리지 않습니다. 성능 수치는 bootstrap 95% CI를 함께 보고하고, validation에서만 model/threshold를 선택합니다.

Baseline ladder

Complexity must earn its place.

첫 논문에서는 가장 복잡한 diffusion 또는 graph model을 바로 주 모델로 두지 않습니다. 같은 split과 same feature availability 아래 단순한 구조적 기준선부터 시작해, 각 단계가 추가 가치를 주는지 ablation으로 확인합니다.

B0 · rule-based

Robust morphometry

normal control의 structure volume·ratio에 대해 age/protocol-conditional robust z-score를 계산합니다. 가장 해석 가능한 reference입니다.

B1 · classical one-class

Mahalanobis / Isolation Forest

segmentation-derived descriptor와 PCA shape embedding에 적용합니다. fit은 normal train controls만 사용합니다.

B2 · learned representation

Contrastive shape encoder

augmentation-consistent 3D embedding을 학습하고 normal density 또는 one-class boundary로 score합니다. image-only score와 shape score를 분리 보고합니다.

B3 · optional

Generative residual model

controls가 충분하고 protocol harmonization이 확인된 뒤에만 masked reconstruction/diffusion residual을 비교합니다. residual image가 artifact를 anatomy로 오인하는지 audit합니다.

Recommended first paper. HVSMR segmentation backbone + interpretable B0/B1 structure score + matched normal controls를 주 분석으로 둡니다. SAD의 memory/pseudo-label mechanism이나 SELSA-style aggregation은 fetal ultrasound video cohort가 확보된 뒤의 별도 연구 질문으로 남깁니다.
