# Dataset audit template

각 데이터셋마다 아래 항목을 채운다.

- 공식 출처/DOI:
- 원 논문/GitHub:
- 공개·비공개·request 여부:
- license/사용 제한:
- 환자 수와 파일 수:
- modality와 파일 형식:
- label ontology 및 annotation type:
- train/validation/test split 방식:
- patient-level leakage 가능성:
- class imbalance:
- preprocessing/augmentation:
- baseline architecture:
- metric와 reported result:
- 샘플 3개 시각화/통계:
- 재현성 위험 및 다음 조치:
