Background knowledge

Before SAD: what makes a graph dynamic?

static graph는 한 번 정한 adjacency matrix를 기준으로 node embedding을 계산합니다. 반면 SAD의 입력은 시간 순서가 있는 event stream입니다. 논문 표기에서 event δ(t) = (vᵢ, vⱼ, t, xᵢⱼ)는 source node, target node, 시각, edge feature를 함께 가리킵니다. 같은 node 쌍도 다른 시각에 여러 번 연결될 수 있습니다.

그래서 “이 node가 누구와 연결됐는가”만으로는 부족합니다. 언제 연결됐는가, 직전 상호작용에서 얼마나 시간이 지났는가, 과거 이웃이 어떤 feature를 가졌는가가 representation의 일부가 됩니다. SAD는 TGAT를 temporal encoder로 사용해 이 과거 이웃을 attention으로 집계합니다.

Temporal encoding

Elapsed time is a feature

TGAT의 relative time encoding은 Δt를 연속적인 representation으로 바꿉니다. 같은 관계라도 최근 관측과 오래된 관측을 다르게 읽게 하는 출발점입니다.

Message passing

Neighbors become evidence

각 layer는 과거 이웃의 embedding, edge feature, time encoding을 aggregate하고 자기 node의 이전 representation과 combine합니다. 이것이 zᵢ(t)를 만듭니다.

Anomaly score

Score is not a probability

MLP가 zᵢ(t)를 실수 score sᵢ(t)로 바꿉니다. SAD에서 중요한 것은 sigmoid 확률이 아니라, 현재 reference distribution에서 표준화했을 때 얼마나 벗어나는가입니다.

One-minute read

What problem does it solve?

SAD는 시간이 흐르며 node, edge, attribute가 변하는 그래프에서 이상을 찾습니다. 충분한 이상 라벨을 모으기 어려운 현실을 전제로, 소수의 라벨과 다수의 미라벨 표본을 함께 이용합니다.

시간 그래프 encoder가 node representation을 만들고, anomaly detector가 점수를 예측합니다. 여기에 시간 정보를 가진 memory bank가 미라벨 표본의 분포를 참조점으로 제공하며, 점수 차이로 만든 의사 그룹이 contrastive learning에 쓰입니다.

Paper scope. 원 논문은 Wikipedia·Reddit·MOOC·Alipay의 동적 상호작용 그래프를 대상으로 합니다. 태아 초음파는 그래프로 주어지지 않으므로, node·edge·시간 정의를 별도로 정당화해야 합니다.

Architecture · original figure

Use time, scarce labels, and unlabeled structure together.

01 · temporal encoder

TGAT representation

과거 이웃의 attribute, edge feature, 상대 시간차를 message passing으로 모읍니다. 논문 구현은 2-layer·2-head TGAT, 128-dimensional embedding입니다.

02 · score head

Deviation network

2-layer MLP가 zᵢ(t)를 scalar anomaly score sᵢ(t)로 바꿉니다. 절대적 score 자체보다, 현재 정상 reference 분포에서의 거리로 판단합니다.

03 · memory prior

Time-decayed FIFO queue

normal·unlabeled score와 저장 시각을 queue에 보관합니다. 오래된 score는 작은 weight를 받고, queue가 차면 FIFO 방식으로 제거됩니다.

04 · pseudo supervision

Score-aware contrast

deviation 차이가 작은 node를 pseudo-positive로 묶어 embedding을 가깝게, 큰 차이는 멀게 만듭니다. 이 loss는 encoder에만 역전파됩니다.

Mechanism in detail

Four coupled components, one objective.

SAD의 설계는 “few-shot label만으로 score head를 학습하면 미라벨 data가 버려진다”는 문제에서 출발합니다. 따라서 미라벨 표본을 (1) 정상 reference distribution의 추정과 (2) representation regularization 두 경로에 사용합니다. 둘은 서로 다른 역할을 하며, 단순 pseudo-label classification과는 다릅니다.

문제 정의에서 event는 δ(t) = (vᵢ, vⱼ, t, xᵢⱼ)입니다. 즉 node 쌍의 interaction과 시각, edge feature가 들어오는 continuous-time graph입니다. 저자들은 단일 static graph snapshot이 아니라 과거 event에 조건화된 node representation을 사용합니다.

Temporal graph encoder

Relative time enters message passing

k번째 layer는 과거 neighbor의 representation, edge feature, 상대 시간 Δt를 aggregate한 뒤 자기 representation과 combine합니다. 논문은 cosine-based relative time encoding을 쓰는 TGAT를 backbone으로 택합니다.

zᵢ(t) = TGAT(G<t, Δt, xᵢⱼ)
Score + memory

Learn a moving reference, not a fixed Gaussian

MLP score head는 sᵢ(t)=W₂ReLU(W₁zᵢ+b₁)+b₂입니다. memory는 y=0(normal) 또는 −1(unlabeled)의 score·timestamp만 저장합니다. 최근 score일수록 큰 time-decay weight로 weighted mean과 standard deviation을 계산합니다.

dev(vᵢ,t) = (sᵢ(t) − μᵣ(t)) / σᵣ(t)
Two losses

Separate labeled deviation from unlabeled contrast

labeled normal은 deviation이 0으로, labeled anomaly는 margin 이상으로 멀어지도록 Ldev를 계산합니다. 미라벨 표본은 deviation 차이가 1σ 미만이면 positive pair가 되며, 가중 supervised contrastive loss Lscl가 encoder를 정렬합니다.

L = Ldev(θenc,θano) + αLscl(θenc)
Modeling assumption. memory bank는 미라벨 표본의 대부분이 정상이라는 가정에 기대며, score distribution을 Gaussian으로 근사합니다. anomaly contamination이 높거나 drift가 매우 빠른 데이터에서는 이 prior가 오히려 score를 왜곡할 수 있습니다. 이 가정은 적용 전에 별도로 검증해야 합니다.

Learning mechanics

Two losses use unlabeled data in two different ways.

SAD가 미라벨 데이터를 “라벨인 것처럼” 바로 분류하는 것은 아닙니다. 먼저 normal과 unlabeled score를 timestamp와 함께 FIFO memory에 저장해 reference distribution을 만들고, 이 분포는 labeled normal/anomaly의 deviation loss에만 기준점을 제공합니다. 다음으로 unlabeled sample끼리는 deviation 차이가 1σ 미만인 경우에만 pseudo-positive pair로 묶어 encoder representation을 정렬합니다.

따라서 score head와 encoder의 역할이 다릅니다. Ldev는 encoder와 anomaly detector를 함께 업데이트하지만, Lscl은 논문 정의상 temporal graph encoder parameter에만 전달됩니다. 이 분리는 pseudo-label이 score head를 직접 끌고 가는 피드백을 줄이려는 설계로 읽을 수 있습니다.

Deviation loss

Anchor the labeled cases

dev=(s−μᵣ)/σᵣ입니다. labeled normal은 |dev|를 작게, labeled anomaly는 margin m보다 크게 만들도록 학습합니다. μᵣ·σᵣ은 고정 standard normal이 아니라 memory에서 추정됩니다.

Time decay

Recent scores count more

논문은 저장 시각이 오래될수록 작은 weight를 사용합니다. 이는 data drift와 학습 중 parameter 변화에 대응하려는 것이지만, decay rate와 memory size는 성능·안정성에 직접 영향을 주는 hyperparameter입니다.

Contrastive pairs

Align only nearby deviations

batch 안의 두 sample deviation 차이가 1σ 미만이면 positive pair가 되고, 차이가 작을수록 큰 weight를 받습니다. representation similarity가 score-space grouping을 강화하지만, 초기 score 오류도 전파될 수 있습니다.

Proposed translation

When is a graph formulation defensible for fetal video?

프레임을 단순한 시간열로 다루는 것만으로 충분하지 않을 때에만 그래프를 고려해야 합니다. 예를 들어 frame token을 node로, temporal adjacency·appearance similarity·anatomy/view consistency를 edge로 둔다면, 품질 변화와 반복 관측을 함께 표현할 수 있습니다.

Node

Frame or region token

한 clip의 frame, 혹은 검출된 cardiac region을 node로 두고 image embedding과 quality feature를 attribute로 둡니다.

Edge

Temporal + semantic relation

시간 인접 edge만이 아니라 view/phase/appearance 유사 edge를 명시하고, edge 생성 규칙을 train set에서 고정합니다.

Label

Screening or review signal

확진 CHD label이 희소하다면 frame label을 억지로 만들기보다 clip/patient label과 review priority를 구분해 설계합니다.

Critical guardrail. 의사라벨은 초기 anomaly score의 오류를 증폭할 수 있습니다. 따라서 confidence threshold, warm-up, class imbalance 처리, patient-level leakage 검사, 임상의 검토 가능한 hard case audit가 사전 조건입니다.

Implementation review

Questions that decide whether SAD belongs in the baseline.

  1. 01

    Do we have a meaningful graph?

    시간 순서만 있다면 temporal transformer/RNN/pooling이 더 단순한 기준선입니다. graph edge가 추가 정보를 주는지 먼저 확인합니다.

  2. 02

    What is “normal”?

    정상 분포의 정의가 기관·주수·view quality에 따라 변할 수 있습니다. memory bank가 섞어서는 안 되는 subgroup을 정합니다.

  3. 03

    How will pseudo-label harm be measured?

    라벨 비율을 단계적으로 줄이는 ablation, calibration, anomaly recall at fixed review budget, 오류 사례 표본 감사를 포함합니다.
